What is HTML

HTML stands for HyperText Markup Language. Developed in 1990, HTML is the "hidden" code that helps us communicate with others in the internet.

Search Engine Optimisation

SEO stands for Search Engine Optimization. It is the process of getting traffic from the search results on search engines. All major search engines such as Google, Bing and Yahoo have primary search results, where web pages and other content such as videos or local listings are shown and ranked based on what the search engine considers most relevant to users.
